# linux-dotfiles

`packages.txt` - a list of packages required for the full operation of configurations.

If using "Arch Linux", you can run the following commands:

```bash
install_pkgs       # installing packages from packages.txt
refresh_mirrorlist # "reflector" is called under the hood
```

---

Available Bash commands:
- `bhelp` - display the `--help` output in a bat-styled format
